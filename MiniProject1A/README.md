<b>Mini-project 1A: (due in 1.5 weeks)  Emotion Classification with Brain Circuit Simulation</b>

This project provides an introduction to a simple workflow implementation on two typical CI platforms using desktop/laptop and FABRIC systems. Students design a system to simulate activity in the brain circuit related to fear (‘lizard brain’ – most primitive) using Python-based brain circuit model on the computing system. In the design, the user can change parameters (input) and the amygala (brain circuit) model will then predict the brain 'state' (no fear and fear, in this case). The workflow design should then send the output to the LEDS on the microbit to reproduce the firing rate of the amygdala - low flashing indicating no fear and rapid flashing indicating fear.  We will add the cutebot and do fancy stuff later, but this one is to get you started with the design of a simple end-to-end CI workflow. As mentioned, you do this using (i) local desktop and (ii) FABRIC platforms, and a brain circuit model (using NEURON) provided to you.
